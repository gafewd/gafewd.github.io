jQuery CSS Selectors

jQuery CSS Selectors - $('x') function

• CSS selectors query HTML for certain elements

• Similar to document.getElementById()

• Use the same selectors you write in CSS!

• The $('x') Function - Get all of the 'X'


	$('div') // select all <div>'s
	$('h2') // select all <h2>'s
	$('p a') // select all <a>'s inside of all <p>'s
	$('p, a') // select all <p>'s and <a>'s

• The $('x') function takes ANY CSS selector

jQuery CSS Selectors - ID's and Classes

Fetch ID's using the # (hashtag):


	$('#someId')
	$('#someId a')

Fetch Classes using the . (period):


	$('.someClass')
	$('.someClass a')

• Using $('.someClass') returns ALL of the HTML elements with a class of .someClass

jQuery CSS Selectors - $()

$() - the almighty selector!

$('div')

⇒ Select all HTML div elements

$('#someId')

⇒ Selects the first HTML with an ID of "someId"

$('.someClass')

⇒ Select all the HTML elements with class "someClass"

$('ul li a.social-link')

⇒ Select anchor tags with class "social-link" that are nested inside <li> elements which are nested inside <ul>

HTML Manipulation

jQuery HTML Manipulation

.html()

Will return any HTML inside the selected element:


	$('#someId').html()

.html('<h1>a heading</h1>')

Inserts and overwrites the HTML inside the selected element:


	$('#someId').html('
a heading
')

• Use instead of the .innerHTML JavaScript method

jQuery HTML Manipulation

.text()

Returns any text inside the selected element


	$('#someId').text()

.text('some string')

Overwrites any text (or HTML!) inside the selected element


	$('#someId').text('some text!')

Click Event

jQuery Click Event - Named Functions

JavaScript:


	document.getElementById('someId').onclick = someFunction

	function someFunction () {
		// do something
	}

jQuery


	$('#someId').click(someFunction)

	function someFunction () {
		// do something
	}

jQuery Click Event - Anonymous Functions

JavaScript:


	document.getElementById('someId').onclick = function () {
		// do something
	}

jQuery


	$('#someId').click(function () {
		// do something
	})

<input> and .val()

<input> and .val()

JavaScript:


	document.getElementById('someId').value
	// returns the value inside fo someId

jQuery:


	$('#someId').val()
	// returns the value inside of someId

<input> and .val()

• Getting values, use .val()


	$('#someId').val()
	// returns the value inside of someId

• Setting values, use .val('some text')


	$('#someId').val('some text')
	// sets the value of someId to 'some text'

.val() can be used on both <input> and <select> elements

Using jQuery

Using jQuery

Option 1: Download and store jQuery locally

1. Download jQuery from http://jquery.com/download

2. Save the .js file in your project's js/ directory

3. Create a <script> tag ABOVE any of your own JS files


	<script type="text/javascript" src="jquery.js"></script>
	<script type="text/javascript" src="project.js"></script>

Using jQuery

Option 2: Link to an external jQuery source (CDN)

1. Locate an externally hosted version of jQuery:

⇒ https://code.jquery.com/jquery-1.11.3.js

2. Create a <script> tag ABOVE any of your own JS files

3. Point the src attribute to the jQuery CDN


<script type="text/javascript" src="https://code.jquery.com/jquery-1.11.3.js"></script>
<script type="text/javascript" src="project.js"></script>

Using jQuery - Script placement

<script> tags should always be placed:

• at the BOTTOM of your <body>

• right before the closing </body> tag

Why:

• As your JS will reference DOM elements, these elements have to be loaded first!

• By placing <script> tags at the BOTTOM, the browser will load these <script>'s LAST!

Using jQuery - $(document).ready()

Best practice is to write jQuery using the $(document).ready() function:


	$(document).ready(function() {

		// WRITE OTHER CODE HERE!!

	})

• Tells jQuery to wait until HTML/CSS have loaded before executing any JavaScript

• Only execute code once the document (DOM) is ready

CSS Manipulation

CSS Manipulation - .css()

JavaScript:


	document.getElementById('someId').style.color = 'red'

jQuery:


	$('#someId').css('color', 'red')

CSS Manipulation - .css()

• Set Single Property: .css(propertyName, value):


	$('p').css('background-color', 'pink')

• Set Multiple Properties:
.css({ property1: value1, property2: value2 })


	$('#someId').css({ 'height': '100px', 'width': '100px' })

• Get Property: .css(propertyName)


	$('#someId').css('height')
