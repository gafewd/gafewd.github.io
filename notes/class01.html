HTML Basics

HTML - History of HTML

• HTML => HyperText Markup Language

• Invented in 1989 by Sir Timothy Berners-Lee

• EVERY website you've ever visited is rendered in HTML

• Web browsers parse (read) HTML and render (display) the results

HTML Tags

HTML - Tags

• Fundamental building-block of HTML is the element

• Think of an element as a structural piece of your site

• Elements consist of :

Starting tag: <p>

Content: text, image, video, other elements, etc.

Ending tag: </p>

• The content is what the user sees

• The tags tell the browser what TYPE of content it is and how to present it

Group "Chunks" of Content

HTML Structure - DOCTYPE


	<!DOCTYPE html>

• Short for "document type declaration"

• ALWAYS the first line of your HTML

• Tells the browser which "flavor" of HTML is being used

• All modern websites should use HTML5 DOCTYPE

Some Doctypes...

1999

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "DTD/xhtml1-strict.dtd">



2012

<!DOCTYPE html>

HTML Structure - <html>

• <html> is HTML...

• All HTML should be contained inside <html></html>

• Tells the browser that all content inside of the <html> tags is HTML

• Within our <html> tags, we have:

<head></head>

<body></body>

HTML Structure - <head>

• <head> - first tag inside <html></html>

• Adds additional, behind-the-scenes-content

• Not displayed; but is machine parsable

• <title> - adds title to page

• <meta> - adds data...about data

• Some meta data: page description, keywords, author

• Where we load our external CSS documents

HTML Structure - <body>

• <body> - second tag inside <html></html>

• Follows <head></head>

• Contains HTML/content that will be displayed to user

HTML Structure - Basic Structure


	<!DOCTYPE html>
	<html>
		<head>
			<title>The Website Title</title>
		</head>

		<body>
			<h1>ALL HTML CONTENT HERE!</h1>
		</body>
	</html>

<img> tags

HTML - <img> tags

• Images are placed using the <img> tag

• The <img> doesn't wrap content, so it doesn't close!

• Requires a src attribute; tells the browser where the source of the image (the image file) is located


	<img src="cookie.png">

HTML - <img> tags; src attribute

<img>'s src attribute can be a relative/absolute path:

Relative:


 <img src="cookie.png">

Absolute:


 <img src="http://guardianlv.com/wp-content/uploads/2013/12/reeves2.jpg">
