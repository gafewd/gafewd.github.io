Toggling Element Visibility

jQuery Hide and Show

.hide()

⇒ Hides elements by setting their inline style to display: none;


	$('p').hide()

.show()

⇒ Reveals elements by setting their inline style to display: block

⇒ If the element was originally inline, then display: inline will be used


	$('p').show()

jQuery Hide and Show (ctn)

• Change the duration by passing an argument

⇒ String:


	$('p').hide('slow')

	$('p').show('fast')

⇒ Number:


	$('p').hide(700)

	$('p').show(300)

jQuery Hide and Show Alternatives

.fadeIn()

.fadeOut()

.slideDown()

.slideUp()

.slideToggle()

• Each take a duration argument:


	$('p').fadeIn('fast')

	$('p').slideUp(900)

Anonymous Functions

Anonymous Functions Review

• Named Functions:


	function doSomething () {
		// run some code...
	}

• Anonymous Functions:


	function () {
		// run some code...
	}

Anonymous Functions

• Many commands, especially events, expect a function as an argument:


	$('button').click(someFunction)

	function someFunction () {
		// Do something here
	}

• Anonymous functions let us use a function without naming and separately defining it:


	$('button').click(function () {
		// Do something here
	})

• This makes anonymous functions good for events: .click(), .keyup(), .mouseover(), etc.

• NOTE: anonymous functions only exist when that bit of code is running

Manipulating Element Attributes

.attr('attribute-name')

⇒ Gets the value of the matched attribute name


	

	$('img').attr('src')
	// returns 'keanu.jpg'

.attr('attribute-name', 'attribute-value')

⇒ Changes the value of 'attribute-name' to 'attribute value'


	$('img').attr('src', 'gary-busey.jpg')

$(this)

$(this) allows us to target specific elements that invoked (or called) a function

• Refactor previous CodePen using $(this):


	$('span').click(function () {
		$(this).css('color', 'red')
	})

• On the click event, only the <span> that invoked the function will turn red

$(this) - More technical explanation

• In JS, functions are called (invoked) by an object

• Whatever object called the function, can be accessed inside the function using this

• this is said to be the "context" of the function

• We can "wrap" this in jQuery: $(this)

• HTML elements are objects, too:


	$('span').click(function () {
		$(this).css('color', 'red')
	})
