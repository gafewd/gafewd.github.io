Control Flow:

⇒ Order in which the computer executes program's statements

⇒ Determines which chunks of code are running at a given time

Conditional Statements

Sequential Control Flow:

• Programs execute line by line, from top to bottom

• Default for most programming languages

Conditional Statements

Sequential Control Flow Example:


    function square () {
        var inputNum = $('input').val()
        var outputNum = inputNum * inputNum

        $('#output').text(outputNum)
    }

1) Define a variable, inputNum, equal to the value of a HTML element

2) Define a variable, outputNum, equal to inputNum squared

3) Write outputNum to a HTML element

Condtional Statements

Conditional Flows JavaScript Syntax:


    var x = 10

    if (x > 5) {
        alert('x is larger than 5!')
    } else {
        alert('x is smaller than 5!')
    }

• Denoted by the if and else keywords

• Begins with a condition: (x > 5)

• If the condition is true, the statment(s) in the "if block" will run

• If the condition is false, the statment(s) in the "else block" will run

Condtional Statements: Logial Operators

• Logical operators:

And ⇒ &&

Or ⇒ ||

• Enable more complex conditions:


    var x = 2;
    var y = 3;

    if (x == 2 && y == 3) {

    }

Condtional Statements: Logial Operators

• Or Operator:


    var x = 2;
    var y = 3;

    if (x == 'cat' || y == 3) {
        console.log('this will print')

        console.log('this will not print')
    }

Loops

Loops:

• Another form of program control flow

• Repeat a series of steps until a certain condition is met

• while

• for

While Loop

• Runs while the condition is true


    var i = 0;

    while (i < 10) {
        console.log(i);
        i = i + 1;
    }

• What happens if the condition is never false?

• INFINITE LOOP!

• Example

For Loop

• Runs for a set number of loops


    for (var i = 0; i < 10; i++) {
        console.log(i);
    }

• Four parts:

1) Initializer/Counter: var i = 0;

2) Condition (test statement): i < 10;

3) Iteration (increase/decrease the counter): i++

4) Loop Body (code that is executed during each loop)

• Example

Loops in Practice

• Loops are EVERYWHERE!

• Applied when some action needs to be repeated:

⇒ Displaying a list of Google search results

⇒ Displaying a list of recent tweets

⇒ Rendering a photo gallery (a list of photos)
