jQuery Hide and Show

.hide()

⇒ Hides elements by setting their inline style to display: none;


	$('p').hide()

.show()

⇒ Revels elements by setting their inline style to display: block

⇒ If the element was originally inline, then display: inline will be used


	$('p').show()

jQuery Hide and Show (ctn)

Change the duration by passing an argument

String:


	$('p').hide('slow')

	$('p').show('fast')

Number:


	$('p').hide(700)

	$('p').show(300)

jQuery Hide and Show Alternatives

.fadeIn()

.fadeOut()

.slideDown()

.slideUp()

.slideToggle()

Each take a duration argument:


	$('p').fadeIn('fast')

	$('p').slideUp(900)

Anonymous Functions

Anonymous Functions

So far, we've been writing and calling functions like this:


	// Event handler responsible for calling the function
	$('button').click(someFunctionName)

	// The Actual Function
	function someFunctionName () {
		// Do something here
	}

• This is an example of a named function

• The function's name is someFunctionName

Anonymous Functions (ctn)

Anonymous Functions:

⇒ Functions that do not have names: 


	function () {
		// Do something here
	}

You've already used one:


	$(document).ready(function () {
		// Your code here
	})

Anonymous Functions (ctn)

• Some commands expect a function as an argument:


	$('button').click(someFunctionName)

	function someFunctionName () {
		// Do something here
	}

• Anonymous functions let us use a function with out naming and separately defining it:


	$('button').click(function () {
		// Do something here
	})

• This makes anonymous functions good for events: .click(), .keyup(), .mouseover(), etc.

• NOTE: anonymous functions only exist when that bit of code is running

$(this)

$(this) allows us to target specific elements that invoked (or called) a function

• Refactor previous CodePen using $(this):


	$('span').click(function () {
		$(this).css('color', 'red')
	})

• On the click event, only the <span> that invoked the function will turn red

$(this) - More technical explanation

• In JS, functions are called (invoked) by an object

• Whatever object called the function, can be accessed inside the function using this

• this is said to be the "context" of the function

• We can "wrap" this in jQuery: $(this)

• HTML elements are objects, too:


	$('span').click(function () {
		$(this).css('color', 'red')
	})
