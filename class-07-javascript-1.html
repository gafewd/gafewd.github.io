Pseudo Code

Like code, but not?

Pseudo Code

• Pseudo Code:

⇒ The process of thinking through a program, step-by-step, without actually writing a line of code

⇒ A written outline of a program that can be converted into code

⇒ Allows the programmer to focus on problem solving, not the precise layout of the code and its syntax

⇒ Don't need to know how to code, to write pseudo code!

Pseudo Code

• Let's write pseudo code for a thermostat

• Things to think about:

⇒ What is goal of our program? What are we trying to solve?

⇒ What are the inputs for our program? What pieces of data will be going into it?

⇒ What is the desired output of our program? Once the program runs, what should happen?

Pseudo Code

determine target_temperature

target_temperature = 72

get current_temperature

current_temperature = 68

if current_temperature < target_temperature

turn_heater_up

if current_temperature >= target_temperature

turn_heater_down

Variables

Variables

• Often, you will want to store, or keep track of, data in a program

• Declare using the var keyword

• Will store the piece of data in memory

• Use the variable's name to "call" it, or retrieve it from memory:


	var x = 2;
	var y = 2;

	x
	=> 2

	x + y
	=> 4

Variables - Naming Rules

• Take care when naming your variables

• Variable names should reflect the data they describe

• Other naming conventions:

⇒ Names should start with a lowercase letter

⇒ Cannot use spaces; use hyphens or underscores instead

⇒ Better yet, use CamelCase:


	var currentTemp = 70;
	var targetTemp = 73;

Variables - Reassignment

1. Create a variable, x, and assign it to 2


	var x = 2;
	> 2

2. Perform some arithmetic on x


	x * 2

Variables - Reassignment

⇒ You must reassign variables to change them


	var x = 2;
	> 2

	x * 2
	> 4

	x
	> 2

	x = x * 2
	> 4

	x
	> 4
