HTML Basics

HTML - History of HTML

HTML => HyperText Markup Language

Invented in 1989 by Sir Timothy Berners-Lee

EVERY website you've ever visited is rendered in HTML

Web browsers parse (read) HTML and render (display) the results

HTML - Say what?

HT => HyperText

HyperText is text which contains links to other texts.

M => Markup

Markup describes a system of describing or annotating text files. Markup describes the structure/layout of a webpage.

L => Language

Just like any language, HTML has its own vocabulary, grammar, and syntax. You can think of HTML as the language that the web speaks.

HTML Tags

HTML - Tags

The fundamental building-block of HTML is the Element. Think of an Element as a structural piece of your site.

Elements in turn consist of starting tags, content, and ending tags.

The content is what is displayed on the website. The tags tell the browser what TYPE of content it is and how to present it.

HTML - Tags (continued)

We use code called "tags" to group content into different chunks.

<p>Hello, World!</p>

There generally is an opening tag <sometag> and a closing tag </sometag> wrapping the chunk of content

Group "Chunks" of Content

HTML Structure - DOCTYPE

<!DOCTYPE html>

Short for "document type declaration"

ALWAYS the first line of your HTML

Tells the browser which flavor of HTML we are using

All modern websites should use HTML5 DOCTYPE

Those DOCTYPES...

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "DTD/xhtml1-strict.dtd">



<!DOCTYPE html>

HTML Structure - <html>

<html> is HTML...

All HTML should be contained inside <html></html>

Tells the browser that all content inside of the <html> tags is HTML

Within our <html> tags, we have <head> and <body> tags

HTML Structure - <head>

<head> - first tag inside <html></html>

Adds additional, behind-the-scenes-content

Not displayed; but is machine parsable

<title> - adds title to page

<meta> - adds data...about data

Some meta data: page description, keywords, author

Where we load our external CSS documents

HTML Structure - <body>

<body> - second tag inside <html></html>

Follows <head></head>

Contains HTML that will be displayed

HTML Structure - Basic Structure


<!DOCTYPE html>
<html>
	<head>
		The Website Title
	</head>

	<body>
		ALL HTML CONTENT HERE!
	</body>
</html>

<img> tags

HTML - <img> tags

Images are placed using the <img> tag

The <img> doesn't wrap content, so it doesn't close!

Requires a src attribute; tells the browser where the source of the image (the image file) is located


;

HTML - <img> tags; src attribute

<img>'s src attribute can be a relative/absolute path

Relative:

<img src="cookie.png">

Absolute:

<img src="https://www.google.com/images/ branding/googlelogo/2x/googlelogo_color_272x92dp.png">
