CSS Positioning

CSS Positioning - Overview

Static


	position: static;

Fixed


	position: fixed;

Relative


	position: relative;

Absolute


	position: absolute;

position: static;

• Default CSS positioning value for ALL elements

• Considered "normal" behavior

• Elements with position: static; are said to NOT be positioned

• Cannot set top, right, bottom, and left properties

position: fixed;

• Positioned relative to the browser window

• Does not move when the browser scrolls

• Can position using top, right, bottom, and left

• Removed from the normal flow of the document

• Used for for persistent navigation: "sticky headers"

Sticky Header Example

• Apply the following CSS rules to your <header>:


 header {
 	 position: fixed;
	 width: 100%;
	 top: 0;
	 height: 50px; // should match the padding-top on body
 }

• Apply the following CSS rules to your <body>:


 body {
	 padding-top: 50px // should match the height of header
 }

position: relative;

How would we create this?

position: relative;

• Positioned relative to its normal position

• Move using top, right, bottom, and left properties

• Note: the original space for the element is preserved

• Used for:

⇒ Overlapping elements

⇒ As a reference for absolutely-positioned elements

• CodePen exercise

position: absolute;

• Positioned relative to either:

⇒ Browser window

⇒ Closest positioned element

• Removed from the normal flow of the document

• Element will collapse to content's width

• Used for:

⇒ Animations (we will see shortly)

⇒ Placing an element within a parent

• CodePen Exercise

position: absolute; & position: relative;

• Use position: absolute; with position: relative; to absolutely position an element within another element

⇒ position: relative; is placed on the parent element

⇒ position: absolute; is placed on the parent element

jQuery .animate()

jQuery .animate()

.animate() performs a custom animation on a set of CSS properties

⇒ Animate height/width from Ypx to Zpx

⇒ Animate an element's postion (move element across screen)

• We can call .animate() on most CSS properties

• Cannot .animate() color-related properties

• See jQuery UI Color Animation plugin

.animate() the following:

• Border:

border; border-width; border-bottom-width; etc.

• Margin:

margin; margin-top; margin-right; etc

• Padding:

padding; padding-top; padding-right; etc

• height/width

• font-size

• top/right/bottom/left

• opacity

jQuery .animate()

.animate() signature:

.animate(CSS-properties, duration)

Example:


	$('#someEl').animate({
		'width': '500px',
		'height': '500px'
	}, 2000)
