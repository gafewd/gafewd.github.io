Function Arguments

Function Arguments

So far, functions have been defined like this:


	function someName () {
		// Body of function here
	}

...and called like this:


	someName()

Many of you have inquired: WTF are the () for?

Function Arugments

Arguments:

Data inputs that are "passed" into a function

• Define a function that will accept an argument:


	function hello (name) {
		alert("Hello, it's " + name)
	}

• Call a function that accepts an argument:


	hello('me')

	// outputs: "hello, it's me"

	hello('Michael')
	hello('Stas')

Function Arguments - Multiple Arugments

• Multiple arguments can be passed in using a comma-separated list

• Define a function with multiple arguments:


	function add (x, y) {
		return x + y
	}

• Call a function with multiple arguments:


	add(2, 2)

	// returns 4

	add(2, 5)
	add(100, 1)

Arguments vs Parameters

• Arguments and parameters are often used interchangeably

• There is a clear distinction:

⇒ Parameters refer to the variables declared in the function definition

⇒ Arugments refer to the data passed into the function

Arguments vs Parameters

Parameters:

• Function add takes two parameters: x and y:


	function add (x, y) {
		return x + y
	}

Arguments:

• Call add with the arguments 5 and 2:


	add(5, 2)

	// returns 7

HTML Insertion/Removal

HTML Insertion/Removal

.append('content')

⇒ Insert "content" at the END of each element


	$('div').append('<p>Appended paragraph</p>')

.prepend('content')

⇒ Insert "content" at the BEGINNING of each element


	$('div').prepend('<p>Prepended paragraph</p>')

HTML Insertion/Removal

	$('#someDiv').prepend('<p>Prepended paragraph</p>')

.before('content')

⇒ Insert "content" AFTER each element


	$('div').after('<p>After paragraph</p>')
